# one - a Rougelike game written in DOS Batch script

The project is as the title sounds, this is an initial demo of 'one' - a game written in the DOS Batch scripting language. Not because DOS scripts are the most efficient tool to build a game, but simply as an experiment to see what is possible with them.

So far, I have had a lot of fun re-learning Batch scripting and adding features into the game.
1. Configure translations with "messages.csv"
2. Configure entity attributes with "entities.csv"
3. C-source files "key.c" and "set_console_mode.c", enable user input and escape sequences for the terminal
4. Game source "one.bat" is what makes the "magic" happen, run this file to start the game

## Starting up

The game can be started either by double clicking the file in the File Explorer, or from the Command Prompt with the following command:
> one.bat
